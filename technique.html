Logiciel de rendu 3D

Unity C#

Pré-requis

  • Unity v2021.3.16F1

Installation

  • Ouvrir le projet dans Unity
  • Exécuter le projet dans Unity

Description

Le logiciel de rendu 3D Urbanext permet de créer les conteneurs, de les assembler et ensuite de les placer dynamiquement dans le quartier afin de les visualiser. Il sera même possible d’imprimer votre prototype en 3D. Notre prototype contient actuellement un module pour créer les conteneurs et les agencer, ainsi qu’une visualisation d’une carte en 3D importée.

Application Urbanext Gestion

Vue TypeScript

Pré-requis

  • Node.js v16.0+

Installation

  • Ouvrir un terminal à la racine du projet
  • npm i
  • npm run dev

Description

L’application Urbanext Gestion permet de gérer tous les services liés à l’immeuble et permet d’avoir également un tableau de bord sur la consommation actuelle de l’immeuble. Notre logiciel actuel permet de gérer la plateforme d’autopartage, notamment en gérant les réservations des voitures. Il est également possible de voir la consommation de nos trajets. Il est également possible d’accéder au service de réservation de laverie et au service d’emprunt d’objets.
